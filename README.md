API](https://help.opendatasoft.com/apis/ods-search-v2/#search-api-v2).

## Installation

You can install the released version of odsapi from
[GitHub](https://github.com/ijlyttle/odsapi) with:

``` r

remotes::install_github("ijlyttle/odsapi")
```
